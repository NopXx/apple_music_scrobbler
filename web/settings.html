Last.fm ให้ฟรี ใช้เวลาไม่ถึงนาที:
  1. กดปุ่มด้านล่างเพื่อเปิดหน้าสมัคร
  2. กรอกชื่อ application (อะไรก็ได้) แล้ว Submit
  3. Copy ค่า API key และ Shared secret มาวางในช่องด้านล่าง
  4. กด "Connect with Last.fm" อีกครั้ง

Webhook
ส่ง POST JSON เมื่อเกิด event: play, replay, pause, resume, scrobble
วินาที — ส่งข้อมูลซ้ำทุกๆกี่วินาที, 0 = ปิด
